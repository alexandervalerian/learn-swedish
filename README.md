# Learn Swedish

Swedish vocabulary flashcard app using spaced repetition (SM-2). Words are organized by CEFR level (A1–C1) with German translations.

## Stack

Nuxt 4 SPA · Vite · Tailwind CSS v4 · Pinia · localStorage

## Setup

```bash
npm install
npm run dev       # localhost:3000
npm run build
npm run preview
```
